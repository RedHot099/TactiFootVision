# Materiały do slajdów

Każdy folder `slide_0X` zawiera:
- podlinkowane assety do tego slajdu,
- plik `copy_pl.md` z dokładnym tekstem do wpisania,
- krótką notatkę o układzie slajdu.

Kolejność:
- `slide_01`: tytuł i problem
- `slide_02`: pipeline
- `slide_03`: homografia
- `slide_04`: demo wideo
- `slide_05`: walidacja end-to-end
- `slide_06`: trade-off jakości i wydajności
- `slide_07`: final output i dalsze kroki
